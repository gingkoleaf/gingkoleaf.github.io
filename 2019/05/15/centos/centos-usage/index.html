yum list

命令 描述
yum list [all] 所有已安装和可以安装的包
yum list available 所有可以安装的包
yum list installed 所有已经安装的包
yum list updates 所有可以升级的包
yum list extra 所有已经安装但不在repository的包

查看软件包详情

yum info wget

查看软件包依赖

yum deplist wget
